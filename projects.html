Predicting Flight Delays

Tools used: Python, Sklearn, Tableau, Myplotlib, Gridsearch

Airline travel can be stressful for various reasons such as weather, technical issues, or delays from arrivals and departures.

My group and I utilized Python techniques and libraries to be able to predict if a flight will be delayed given a set of features. We considered a delay to be if the flight took off more than fifteen past the original departure time.

After cleaning the data, we conducted EDA to better understand the data to find any correlations or any notable characteristics in the data. We applied binarization to simplify columns to “Was there [...]” or “Was there no [...]” for various weather features. In addition we applied Integer encoding for time blocks.

After the preprocessing, we ran a gridsearch in order to find the best parameters for Logistic Regression and Decision Tree. We also used ensemble models in an effort to get a better F1 score.

Results: We found that XGBoost Trees had the best F1 Score of 0.39887.
